So why Helix?

Managing a website is complex. Popular CMS platforms - such as Wordpress, Drupal, and Expression Engine - offer many features and extensions but at the cost of ease of use and simplicity. We don’t think that should be the case. That’s why we built Helix.

Content editing, easy as pie

Writing for the web can be difficult with so many technologies and coding languages dictating how your content is presented. Helix allows website editing to be easy and understandable, we made it work just like a text editor you would use to write a paper.

Using Helix is easy because you don’t have to worry about tags or code when writing. Focus on the message of the site and leave the heavy lifting to Helix.

Drag & Drop Image Editing

Text can be boring, so we recommend adding plenty of great images to spice up a website. To make adding images easy we built the Image Bank. The Image Bank stores your images in a tabbed shelf on the side of your site for quick easy access.

And to make it better, we added drag and drop functionality to make inputing your images simple.

No admin, no templates, no code

We believe that anyone should be able to edit and update their website anywhere, anytime. And be able to do so without having knowledge of coding languages, sifting through administrative dashboards or dealing with user permissions.

Helix takes your website, built using good ol’ HTML, CSS, and JS, adds a bit of magic, and voila, your website becomes easier to manage than using a site built on a CMS platform.

Save time, save money

Static websites offer quick, easy development but lack the ability to easily update and edit content. Dynamic websites offer editing functionality but at the cost of complexity and extra time and money to develop.

Helix offers the advantages of both. Save time and money by developing a static site, but add in the content management capabilities of dynamic websites.
